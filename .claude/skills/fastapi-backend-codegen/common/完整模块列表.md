# 公共模块完整列表

本文档列出所有公共模块及其在实际项目中的位置。

## 核心模块

### 1. 数据库连接 (`db.py`)
- **实际位置**: `app/db/sqlalchemy.py`
- **主要导出**: `async_session`, `async_transaction`, `content_transaction`
- **用途**: 数据库连接和事务管理

### 2. Redis连接 (`redis.py`)
- **实际位置**: `app/db/redis.py`
- **主要导出**: `redis_pool`, `MyAsyncRedis`, `RedisPool`
- **用途**: Redis连接和操作（自动JSON序列化）

### 3. 配置管理 (`config.py`)
- **实际位置**: `app/config.py`
- **主要导出**: `Settings`, `settings`
- **用途**: 应用配置管理（从.env文件读取）

### 4. API依赖注入 (`deps.py`)
- **实际位置**: `app/api/deps.py`
- **主要导出**: `get_db`, `get_redis`
- **用途**: FastAPI依赖注入函数

### 5. API路由聚合 (`api_router.py`)
- **实际位置**: `app/api/api_router.py`
- **主要导出**: `router`
- **用途**: 聚合所有API路由

## 异常和工具

### 6. 异常类定义 (`exceptions.py`)
- **实际位置**: `app/exceptions/exceptions.py`
- **主要导出**: 
  - `BusinessException` - 业务异常
  - `IdNotExist` - ID不存在
  - `ValidationException` - 验证异常
  - `ParameterError` - 参数错误
  - `SystemException` - 系统异常
  - `DatabaseException` - 数据库异常
  - `AuthenticationException` - 认证异常
  - `AuthorizationException` - 授权异常
- **用途**: 统一的异常处理

### 7. 核心库说明 (`corelibs.py`)
- **实际位置**: `app/corelibs/`
- **包含模块**:
  - `custom_router.py` - 自定义路由（自动设置请求上下文）
  - `logger.py` - 日志模块
  - `codes.py` - 状态码定义
  - `local.py` - 本地上下文变量
  - `consts.py` - 常量定义
- **用途**: 核心功能库

### 8. 工具函数说明 (`utils.py`)
- **实际位置**: `app/utils/`
- **包含模块**:
  - `response.py` - 响应工具（HttpResponse）
  - `context.py` - 上下文管理（AppTraceId等）
  - `serialize.py` - 序列化工具
  - `common.py` - 通用工具函数
  - `current_user.py` - 当前用户工具
- **用途**: 通用工具函数

## 使用原则

1. **所有模块已存在**：这些模块在实际项目中已经存在，生成代码时直接导入使用即可

2. **不需要复制代码**：不需要将common目录下的代码复制到项目中，它们只是参考

3. **直接导入使用**：生成代码时直接使用导入语句，如：
   ```python
   from app.models.base import Base
   from app.utils.response import HttpResponse
   ```

4. **遵循架构规范**：使用这些模块时遵循项目的架构规范（Model-Service-API分层）

## 模块依赖关系

```
main.py
  ├── init_router (from init/routers.py)
  │     └── api_router (from api/api_router.py)
  ├── init_exception (from init/exception.py)
  │     └── exceptions (from exceptions/exceptions.py)
  ├── init_cors (from init/cors.py)
  ├── redis_pool (from db/redis.py)
  └── init_logger (from corelibs/logger.py)

Model层
  └── Base (from models/base.py)
        ├── 公共字段（id, creation_date, created_by等）
        ├── CRUD方法（get, create, update, delete等）
        └── async_transaction (from db/sqlalchemy.py)

Service层
  └── Model层
  └── exceptions (from exceptions/exceptions.py)
  └── redis_pool (from db/redis.py)

API层
  └── Service层
  └── APIRouter (from corelibs/custom_router.py)
  └── HttpResponse (from utils/response.py)
  └── Schema (from schemas/common.py)
```
