# 数据三级分类



## 🚫 严禁发送（Critical）

以下数据绝对禁止发送给 AI 或暴露在代码中：

| 数据类型 | 示例 | 处理要求 |
|----------|------|----------|
| 数据库连接密码 | `jdbc:mysql://user:pass@host/db` | 使用环境变量 |
| API 密钥 | `sk-xxxx`, `AKIAxxxxxxxxxxxxxxxx` | 使用密钥管理服务 |
| Token | `Bearer eyJhbGciOiJIUzI1NiJ9...` | 不硬编码，运行时获取 |
| 用户隐私数据 | 姓名、手机号、邮箱、身份证号 | 脱敏或使用测试数据 |
| 内部通讯记录 | 钉钉/企微聊天记录 | 不发送 |
| 财务敏感数据 | 金额、账户、交易记录 | 脱敏或使用测试数据 |
| 私钥证书 | `-----BEGIN PRIVATE KEY-----` | 使用密钥管理服务 |

## ⚠️ 脱敏后发送（Warning）

以下数据需脱敏处理后方可使用：

| 数据类型 | 脱敏方式 | 示例 |
|----------|----------|------|
| 日志文件 | 脱敏用户名、ID | `张三` → `***`, `uid=12345` → `uid=***` |
| 错误信息 | 隐藏敏感路径 | `/home/admin/app/config` → `/home/***/app/config` |
| 业务数据 | 使用测试数据替代 | 真实订单 → Mock 订单 |
| 内部 IP | 掩码处理 | `192.168.1.100` → `192.168.x.x` |

## ✅ 可以发送（Safe）

以下数据可直接使用：

| 数据类型 | 说明 |
|----------|------|
| 脱敏后的代码 | 已移除敏感信息的源码 |
| 设计文档 | design.md、架构图等 |
| 技术问题描述 | Bug 描述、需求说明等 |
| 公开的 API 文档 | Swagger/OpenAPI 文档 |
| 测试数据 | 专门的测试数据集 |
