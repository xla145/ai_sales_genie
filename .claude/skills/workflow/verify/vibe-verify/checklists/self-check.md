# AI 生成代码自检清单



## 功能性

- [ ] 代码逻辑是否符合设计文档？
- [ ] 边界条件是否处理完备？
- [ ] 异常分支是否覆盖？
- [ ] 输入输出是否符合接口契约？

## 质量性

- [ ] 是否符合团队编码规范？
- [ ] 命名是否清晰一致？
- [ ] 是否有不必要的复杂度？
- [ ] 是否有重复代码？

## 安全性

- [ ] 是否有 SQL 注入风险？
- [ ] 是否有 XSS 风险？
- [ ] 敏感数据是否脱敏？
- [ ] 权限校验是否完整？

## 性能性

- [ ] 是否有明显的性能瓶颈？
- [ ] 数据库查询是否优化？
- [ ] 是否有内存泄漏风险？

## 测试性

- [ ] 是否提供单元测试？
- [ ] 测试覆盖是否充分？
- [ ] 边界条件是否有测试？
