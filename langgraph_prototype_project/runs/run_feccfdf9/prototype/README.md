# 智能物流管理系统 - 原型说明

## 1. 原型概述

本原型为智能物流管理系统第三阶段产出物，基于第二阶段系统功能设计与页面详细设计生成的可交互静态原型。

### 1.1 系统信息

| 项目 | 内容 |
|------|------|
| 系统名称 | 智能物流管理系统 |
| 系统类型 | B端业务系统 / 管理后台 |
| 所属行业 | 物流运输行业 |
| 系统形态 | Web管理系统 |
| 目标用户 | 物流公司管理人员、调度员、运营人员、运输人员、车辆管理人员 |

### 1.2 核心业务功能

- 订单录入与跟踪
- 车辆与司机管理
- 智能派单与路线规划
- 运输实时状态监控
- 异常预警与处理
- 签收回单管理
- 物流数据统计分析

## 2. 使用方式

### 2.1 打开方式

1. 直接用浏览器打开 `prototype/index.html` 登录页
2. 默认登录账号：运营人员（可切换角色体验不同权限）
3. 登录后可访问系统各个功能模块

### 2.2 导航方式

- 左侧边栏：主导航菜单，支持折叠
- 顶部面包屑：页面层级导航
- 页面内跳转：点击表格行、按钮、快捷入口等

### 2.3 交互说明

- 所有核心操作按钮可点击演示
- 弹窗、抽屉、表单可真实交互
- 列表筛选、搜索、分页功能可用
- 状态变更、提示反馈完整

## 5. 主要业务链路

### 5.1 订单全流程链路

```
登录 → 首页 → 新建订单 → 订单列表 → 订单详情 → 智能调度 → 运输轨迹 → 订单详情(签收) → 完成
```

### 5.2 智能调度链路

```
登录 → 首页 → 智能调度 → 选择订单 → 智能派单/手动派单 → 确认派单 → 司机通知(模拟) → 运输轨迹
```

### 5.3 异常处理链路

```
登录 → 首页 → 异常处理 → 异常详情 → 处理异常 → 异常恢复 → 订单继续执行
```

### 5.4 数据统计链路

```
登录 → 首页 → 数据看板 → 选择统计维度 → 查看报表 → 导出数据
```

## 9. 静态原型限制说明

本原型为静态 HTML 原型，与生产系统存在以下差异：

1. **数据持久化**：所有数据为模拟数据，刷新页面后恢复初始状态
2. **实时更新**：运输轨迹、车辆位置为模拟数据，非真实实时定位
3. **后端验证**：表单校验为前端模拟，后端校验逻辑未实现
4. **通知推送**：司机通知、异常预警为弹窗模拟，非真实推送
5. **地图服务**：轨迹地图使用静态地图或简化版，非真实地图API
6. **权限控制**：权限验证为前端模拟，生产环境需后端配合
7. **导出功能**：数据导出示 PDF/Excel 格式文件
