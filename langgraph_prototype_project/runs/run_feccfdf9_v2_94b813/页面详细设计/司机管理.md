## 2. 页面目标与使用场景

- **页面目标**：展示所有司机信息，支持按状态、准驾车型等条件筛选和搜索，支持新增、查看详情、编辑、删除操作
- **使用场景**：调度员需要查看可用司机、选择司机进行派单、管理司机信息时访问此页面
- **触发入口**：
  - 侧边栏菜单「司机管理」点击进入
  - 智能调度页面点击「选择司机」按钮跳转
  - 首页快捷入口「司机管理」点击进入
- **输出结果**：展示司机列表数据，支持筛选后的结果展示

## 9. 交互逻辑

### 9.1 页面初始化

- **默认展示**：进入页面后，默认加载全部司机列表，按创建时间倒序排列
- **默认筛选**：默认不进行任何筛选，展示全部状态、全部车型的司机
- **默认排序**：按创建时间降序排列，最新创建的司机显示在前
- **默认状态**：每页显示10条数据
- **初始化异常**：加载失败时显示错误提示和重试按钮

### 9.2 查询与筛选

- **筛选项**：
  - 状态下拉框：空闲、工作中、休息中、休息（可选全部）
  - 准驾车型下拉框：A1、A2、A3、B1、B2、C1、C2（可选全部）
  - 搜索框：支持按司机姓名或联系电话模糊搜索
- **查询规则**：筛选条件为"全部"时不作为筛选条件，其他选项精确匹配
- **重置规则**：点击重置按钮后，所有筛选项恢复为"全部"，搜索框清空，列表刷新为全部数据
- **无结果状态**：筛选后无符合条件的数据时，显示空状态区域
- **查询异常**：网络错误时显示错误提示，提供重试按钮

### 9.3 新增/编辑/删除

- **新增逻辑**：
  1. 点击「新增司机」按钮，右侧滑出新增抽屉
  2. 抽屉标题显示为「新增司机」
  3. 表单包含：姓名、联系电话、身份证号、准驾车型、驾驶证有效期、所属公司、所属部门
  4. 所有带*的字段为必填项
  5. 点击「提交」按钮后进行表单校验
  6. 校验通过后调用新增接口
  7. 接口成功后关闭抽屉，刷新列表，显示成功提示
  8. 校验失败时在对应字段下方显示错误提示

- **编辑逻辑**：
  1. 点击操作列的「编辑」按钮，右侧滑出编辑抽屉
  2. 抽屉标题显示为「编辑司机」
  3. 表单预填当前司机信息
  4. 状态为"工作中"的司机不允许编辑，提示"司机工作中，无法编辑"
  5. 点击「提交」按钮后进行表单校验
  6. 校验通过后调用更新接口
  7. 接口成功后关闭抽屉，刷新列表，显示成功提示

- **删除逻辑**：
  1. 点击操作列的「删除」按钮，弹出确认对话框
  2. 确认对话框内容："确定要删除司机{姓名}吗？"
  3. 状态为"工作中"的司机不允许删除，提示"司机工作中，无法删除"
  4. 点击确认后调用删除接口
  5. 接口成功后刷新列表，显示成功提示
  6. 点击取消则关闭对话框，无任何操作

### 9.4 详情与跳转

- **详情入口**：点击表格行或操作列的「查看」按钮
- **跳转目标**：跳转到司机详情页 /driver/detail/:id
- **返回规则**：详情页有「返回」按钮，点击后返回司机管理列表页
- **面包屑规则**：司机管理 > 司机详情

## 15. 原型生成说明

- **页面布局建议**：
  - 顶部区域高度64px，包含页面标题（左侧）和新增按钮（右侧）
  - 筛选区域高度56px，三行筛选项从左到右依次排列
  - 表格区域自适应剩余高度
  - 底部固定分页区域，高度48px

- **关键交互优先级**：
  1. 表格数据加载和展示（高优先级）
  2. 筛选和搜索功能（高优先级）
  3. 新增/编辑/删除操作（高优先级）
  4. 分页功能（中优先级）

- **需要重点表达的业务逻辑**：
  - 司机状态的颜色区分（空闲青色、工作中蓝色、休息灰色）
  - 编辑/删除操作对司机状态的校验逻辑
  - 表单必填项和校验规则

- **可使用模拟数据**：
  - 准备15-20条模拟司机数据，包含不同状态和准驾车型
  - 确保每种状态至少有2-3条数据
  - 模拟数据示例：
    - 张三，13800138000，空闲，A2
    - 李四，13800138001，工作中，B1
    - 王五，13800138002，休息中，A1
    - 赵六，13800138003，空闲，C1
    - 钱七，13800138004，空闲，B2

- **需要避免的原型误差**：
  - 状态标签颜色必须与全局规范一致
  - 操作按钮文字为「查看」「编辑」「删除」，不要用图标代替
  - 删除操作必须有确认对话框
  - 编辑/删除非空闲状态司机时必须有提示
  - 抽屉从右侧滑出，宽度600px
